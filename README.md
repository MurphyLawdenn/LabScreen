LabScreensApp — Лабораторна робота №2
Виконані завдання
У межах роботи було реалізовано:
1.  **Списки (LazyColumn)**: Відображення списку елементів з оптимізацією.
2.  **Навігація (Navigation Compose)**: Перехід між головним екраном та екраном деталей із передачею параметрів.
3.  **AppBar**: Використання `CenterAlignedTopAppBar` та `TopAppBar`.
4.  **Темізація**: Повна підтримка Material 3.

Як запустити:
1. Клонуйте репозиторій.
2. Відкрийте проєкт у Android Studio.
3. Дочекайтеся завершення Gradle Sync.
4. Запустіть застосунок на емуляторі (API 26+) або фізичному пристрої.
