# LabScreensApp — Лабораторна робота №2

Цей проєкт виконано в межах дисципліни «Розробка мобільних застосунків». 

## Тема
**Побудова екранів: списки, деталі, AppBar, темізація (Material 3)**

## Мета
Навчитися створювати екрани у Jetpack Compose з використанням списків (`LazyColumn`), екрану деталей, верхньої панелі (`AppBar`) та застосуванням темізації (`Material 3`).

## Виконані завдання
У межах роботи було реалізовано:
1.  **Списки (LazyColumn)**: Відображення списку елементів з оптимізацією.
2.  **Навігація (Navigation Compose)**: Перехід між головним екраном та екраном деталей із передачею параметрів.
3.  **AppBar**: Використання `CenterAlignedTopAppBar` та `TopAppBar`.
4.  **Темізація**: Повна підтримка Material 3.

### Додаткові завдання (виконано):
*   ✅ **Іконки**: Кожен елемент списку має власну іконку.
*   ✅ **Темна тема**: Реалізовано `darkColorScheme`, застосунок автоматично підтримує нічний режим.
*   ✅ **Меню**: В `AppBar` додана кнопка виклику меню (`Icons.Default.Menu`).
*   ✅ **Поділитися**: На екрані деталей додана кнопка "Share".
*   ✅ **Пошук**: Реалізовано функціонал пошуку/фільтрації списку в реальному часі.

## Технології
*   **Jetpack Compose**
*   **Material 3**
*   **Navigation Component**
*   **Kotlin**

## Як запустити
1. Клонуйте репозиторій.
2. Відкрийте проєкт у Android Studio.
3. Дочекайтеся завершення Gradle Sync.
4. Запустіть застосунок на емуляторі (API 26+) або фізичному пристрої.
